# NCF-007 Agent Standard v1.0

## Status

Approved / Ratified

## Constitutional Definition

An NCF Agent is a governed organizational entity operating under delegated authority for the purpose of achieving one or more measurable business outcomes.

## Mandatory Agent Components

- Identity
- Mission
- Ownership
- KPI
- Authority Class
- Memory Profile
- Tool Profile
- Workflow Portfolio
- Escalation Rules
- Reporting Requirements

## Agent Types

- Executive Agent
- Department Agent
- Specialist Agent
- Workflow Agent
- Utility Agent
- Constitutional Utility Agent

## Agent Lifecycle

Draft → Testing → Approved → Active → Suspended → Retired

## Autonomy Levels

1. Assistant
2. Operator
3. Specialist
4. Manager
5. Executive

## Agent Responsibilities

- Accuracy
- Traceability
- Compliance
- Logging
- Escalation
- Outcome Achievement

## Prime Directive

An Agent is defined by identity, mission, authority, accountability, and outcomes — not by the model it uses.
