# NCF-010 Certification Standard v1.0

## Status

Approved / Ratified

## Purpose

Defines certification requirements, compliance levels, audits, evidence requirements, and revocation procedures.

## Certification Levels

1. NCF-Compatible
2. NCF-Compliant
3. NCF-Verified
4. NCF-Certified

## Certification Targets

- Organizations
- Agents
- Workflows
- Tool Integrations
- Memory Systems
- Data Architectures
- Reference Implementations

## Evidence Examples

- Architecture Diagrams
- Registry Schemas
- Workflow Maps
- Audit Logs
- Run Records
- KPI Reports

## Compliance Scoring

90-100 Certified
75-89 Verified
60-74 Compliant
Below 60 Compatible

## Revocation Triggers

- Constitutional Violations
- Governance Violations
- Security Violations
- Fraudulent Claims

## Prime Directive

Certification provides evidence that an implementation conforms to NCF standards.
