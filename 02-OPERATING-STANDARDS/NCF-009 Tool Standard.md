# NCF-009 Tool Standard v1.0

## Status

Approved / Ratified

## Constitutional Definition

A Tool is an external capability used by an agent to perform work.

## Tool Categories

- Communication
- Storage
- Publishing
- Development
- Finance
- Analytics
- Research
- Operations

## Tool Authority Levels

- T1 Read Only
- T2 Create
- T3 Modify
- T4 Execute
- T5 Administrative

## Tool Governance Principles

- Tools are resources.
- Agents receive permissions.
- Every tool action must be traceable.
- Every tool action must be attributable.
- Every tool action must be auditable.

## Credential Governance

Credentials belong to organizations, not agents.

Credentials must be encrypted, rotated, audited, and scoped.

## Prime Directive

Agents perform work. Workflows organize work. Tools execute work.
