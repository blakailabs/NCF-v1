# NCF-RI-001B Data & Registry Mapping v1.0

## Purpose

Defines how Infinity Loop maps agents, workflows, runs, assets, approvals, memory, events, errors, and metrics into NCF registries.

## Registries

- Organization Registry
- Agent Registry
- Workflow Registry
- Run Registry
- Object Registry
- Asset Registry
- Memory Registry
- Approval Registry
- Error Registry
- Event Registry
- Metrics Registry

## Content Lineage

Idea → Schedule → Production → Publishing → Performance

## Audit Requirements

The system must be able to answer:

- What was published?
- Who generated it?
- Which workflow executed it?
- Which approval authorized it?
- Which metrics resulted from it?
