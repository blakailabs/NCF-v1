# NCF Glossary

## Purpose

Defines core terminology used throughout the Nexus Constitutional Framework.

## Terms

### NCF
Nexus Constitutional Framework.

### CID
Constitutional Identifier. Human-readable identifier that conveys context.

### UID
Universal Identifier. Globally unique machine-readable identifier.

### Agent
A governed organizational entity operating under delegated authority for measurable outcomes.

### Workflow
A governed sequence of activities performed by agents to achieve an outcome.

### Tool
An external capability used by agents to perform work.

### Memory
Governed organizational context used to preserve meaning and continuity.

### Registry
Authoritative system of record for NCF objects.

### Reference Implementation
A canonical example showing how NCF standards are applied in practice.
