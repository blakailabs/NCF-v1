# NCF KPI Catalog

## Purpose

Defines standard KPI measurements used by NCF-compliant workforces.

## Workforce KPIs

- Workflow Success Rate
- Publication Accuracy
- Asset Traceability
- Compliance Rate
- Approval Compliance

## Content KPIs

- Views
- CTR
- Watch Time
- Shares
- Comments
- Saves
- Conversions

## Operational KPIs

- Queue Depth
- Retry Rate
- Error Rate
- Resolution Time
