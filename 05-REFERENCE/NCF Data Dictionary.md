# NCF Data Dictionary

## Purpose

Defines canonical data elements used throughout NCF.

## Core Fields

| Field | Description |
|---|---|
| cid | Constitutional Identifier |
| uid | Universal Identifier |
| parent_cid | Parent lineage identifier |
| lineage_root_cid | Root lineage identifier |
| organization_code | Registered namespace |
| environment | DEV, TEST, STAGE, PROD |
| created_at | Creation timestamp |
| updated_at | Update timestamp |
| schema_version | Contract version |

## Date Context Fields

- date

- week_of_year
- month
- quarter
- year

- timestamp
